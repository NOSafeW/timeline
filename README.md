# SafeW / SafeX Timeline

本仓库按时间整理 SafeW / SafeX 相关公开记录，方便用户快速理解“被点名、下架、改名、再次被点名”的过程。

## 核心时间线

- 2025 年 2 月：Kaspersky Securelist 报告公开 SparkCat 活动，SafeW 相关包名 / Bundle ID 出现在受影响应用列表中。
- 2026 年 1 月：多个中文 Telegram 群 / 频道中再次出现对 SafeW 安全问题的公开讨论和曝光。
- 2026 年 4 月：The Hacker News 报道新一轮 SparkCat 活动，列出 iOS 的 SafeW - 云办公助理与 Android 的 SafeX。
- 2026 年 6 月：NoSafeW 站点和 GitHub 证据仓库整理公开证据，面向中文用户做安全提醒。
